La clase Main contiene el método main el cual genera el proyecto,
se debe cambiar la dirección del método  cargarArchivo() de la clase
ProjectDeductiveDatabase dando la dirección donde se encuentra dlv.bin,
los archivos guessFacts.txt y guessRules.txt.
En la clase interfaz en el método addComponents se debe cambiar la dirección de la imagen en la linea 58 si es que no aparece en la interfaz.
Para este proyecto se usaron las clases proporcionadas en laboratorio y la librería 
DLVwrapper-v4.2, la cual es necesaria para ejecutar el programa.
